Vizsgaremek 

Ez a repository a Ipari informatikus technikus képzés keretében elkészített vizsgaremekemet tartalmazza. A projekt egy tisztán kliensoldali webes alkalmazás.
Projekt leírása:
A weboldala a vizsgakövetelmények alapján épül fel ami a 2021-2026 ig tartó szakmai útamat fejlődéseimet és mérföldköveimet tartalmazza.

Főbb funkciók

    Dinamikus tartalomkezelés JavaScript segítségével.

    Reszponzív felépítés, amely mobil és asztali nézeten is optimális megjelenést biztosít.

Felhasznált technológiák

    HTML5: A weboldal szerkezeti felépítéséhez.

    CSS: Megjelenés, elrendezés (Flexbox/Grid) és animációk.

    JavaScript (ES6+): Dinamikus működés és eseménykezelés.

Telepítés és futtatás

Mivel a projekt nem igényel szerveroldali környezetet, a futtatáshoz kövesse az alábbi lépéseket:

    Klónozza a repository-t:
    git clone https://github.com/Matthew1956/vizsgaremek_share

    Navigáljon a projekt mappájába:
    cd projekt-neve

    Indítás:
    Nyissa meg az vizsgaremek.html fájlt bármilyen modern böngészőben (Chrome, Firefox, Edge).
